## 🚀 About Me

I'm a data engineer who loves building reliable pipelines that power analytics and AI. My day-to-day involves wrangling data across AWS and GCP—whether that's ETL jobs in Glue, transformations in PySpark, or keeping a production pipeline humming. Python is my go-to, and I enjoy automating the boring stuff so data actually ends up where it needs to be. When I'm not coding, you'll find me enjoying Japanese idol wotagei—a unique fusion of music, dance, and lights!

## 🔨 Currently Building

**Aurora → BigQuery Data Pipeline**
Building and maintaining a pipeline that moves data from AWS Aurora (MySQL) to Google BigQuery. Involves orchestrating Glue jobs, managing infra with Terraform, and making sure data lands clean and on time.
